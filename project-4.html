Gmail Inbox Cleaner

A Python automation script that uses the Gmail API to identify, label, and bulk-remove clutter from a Gmail inbox — reducing manual effort and keeping communication organized.

Back to Projects

Project Overview

Email overload is a real productivity problem for small business owners and professionals. This project addresses that by building a Gmail Inbox Cleaner — a Python script that connects to a Gmail account through the Gmail API (OAuth 2.0) and automates the process of finding and removing unwanted messages.

The script can identify emails matching configurable criteria such as sender address, subject keywords, or age threshold, then batch-delete or archive them. This eliminates hundreds of manual clicks and turns a recurring chore into a single command.

How It Works

The tool authenticates with Gmail using OAuth 2.0 credentials and queries the mailbox using Gmail's search syntax (e.g., older_than:30d, from:newsletter@example.com, label:promotions). Matching message IDs are collected in batches and sent to the Gmail API's batch-delete endpoint, keeping API quota usage within limits.

A dry-run mode lets users preview exactly which messages will be affected before any deletion occurs, reducing the risk of accidental data loss. Results are logged to a timestamped file for review.

Key Features

Configurable search filters (sender, keywords, date range), batch processing to stay within API rate limits, dry-run preview mode, run logging, and a straightforward configuration file so non-technical users can customize targets without editing code.

What This Project Demonstrates

This project shows my ability to work with third-party APIs and OAuth authentication, automate repetitive business tasks through scripting, and design practical tools with safety guardrails built in. It reflects a core principle in my work: reducing manual effort without introducing new risks.
